## ðŸ’° Token Management

### Get User Status
Returns balance, plan, and access level.
- **Endpoint**: `GET /user/status`

### Estimate Cost
Project the token cost of a specific mission objective.
- **Endpoint**: `POST /estimate`
- **Body**: `{ "objective": "string" }`

### Admin: Top Up Tokens (Privileged)
Atomic recharge of a user's token balance.
- **Endpoint**: `POST /admin/topup`
- **Role Required**: `ADMIN`
- **Body**: `{ "user_id": "clerk_user_id", "amount": 10000 }`

### Stream Mission
Trigger a tactical mission with real-time SSE progress updates.
- **Endpoint**: `POST /execute/stream`

- **Response**: `text/event-stream`
